# Prototype Code Rules

When editing prototype code:

1. **README required**: Every prototype directory MUST have a README.md with: Hypothesis, How to Test, Findings.

2. **Relaxed standards**: Code quality is relaxed — prototypes are throwaway.

3. **Isolation**: Prototype code must NEVER be imported into production. Rewrite from scratch based on learnings.

4. **Scope**: Keep prototypes minimal — test one hypothesis at a time.
