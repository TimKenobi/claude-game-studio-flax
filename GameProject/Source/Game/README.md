This directory will contain C# gameplay scripts organized by system.

## Directory Structure

```
Game/
  Core/          — Core systems (GameManager, ServiceLocator, EventBus)
  Player/        — Player controller, camera, input handling
  Combat/        — Combat system, weapons, abilities, damage
  AI/            — Enemy AI, behavior trees, pathfinding
  UI/            — HUD, menus, dialogs, inventory screens
  World/         — Level management, environment interactions
  Narrative/     — Dialog system, cutscenes, quest tracking
  Audio/         — Audio manager, sound system, music
  VFX/           — Visual effects controllers
  Utility/       — Helpers, extensions, object pooling
```

Each system directory will contain C# Script classes following Flax Engine patterns.
