# Design — Game Design Documents
This directory contains all game design documentation.

## Structure

```
gdd/
  game-concept.md      — High-level game concept (created by /brainstorm)
  game-pillars.md      — Design pillars
  systems-index.md     — Systems map with priorities (created by /map-systems)
  systems/             — Per-system GDDs (created by /design-system)
narrative/             — Story, dialog, lore documents
levels/                — Level design documents
art/                   — Art bible, asset specifications
```
